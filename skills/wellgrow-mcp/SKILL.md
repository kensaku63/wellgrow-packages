# WellGrow MCP — パーソナルナレッジベース活用

WellGrow MCP を通じて、ユーザーの質問・回答データベース（パーソナルナレッジベース）にアクセスできる。
ユーザーが自分自身で言語化してきた考え・価値観・経験・知識を理解し、**その人らしい** 支援を提供する。

## ツール一覧

| ツール | 用途 | 読み書き |
|--------|------|----------|
| `search_user_context` | 質問・回答をセマンティック検索 | 読み取り |
| `list_questions` | 質問一覧を取得（タグ・ステータスでフィルタ可） | 読み取り |
| `answer_question` | 質問に対して回答を書き込む | 書き込み |

## いつ検索するか

- **会話の開始時** — `list_questions`を使ってピンが立てられた重要な問いを把握する。
- **新しい話題や固有名詞が登場した時** — `search_user_context`を使って関係する質問・回答を取得する。積極的に利用してユーザーの理解を深めながら会話をするようにする。

### 検索のコツ

- クエリには「質問形式」で入力する。
- キーワードには固有名詞や関係しそうな重要キーワードを入力する。

## いつ`answer_question`を使うか。

- ユーザーがピンを立てた問いについては、常に、いい提案ができないかを考える。
- ユーザーにとって新しい発見や気づきがありそうなものを見つけたら提案する。
- ユーザーがなにか気づきや発見を得たと判断したときは、必ず、問いに紐づけて提案する。
